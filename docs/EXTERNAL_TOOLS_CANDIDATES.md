## What to evaluate when the list is available
For each app/tool, classify it into one of these buckets:

1. **Research augmentation**
   - macro/economic calendar
   - fundamentals
   - screening
   - news/sentiment

2. **Signal validation support**
   - backtesting
   - chart replay
   - breadth/market internals
   - options/open-interest/flow

3. **Operator support**
   - watchlists
   - alerts
   - journaling
   - paper trading

4. **Reject / low value**
   - duplicate functionality
   - poor exportability
   - weak data quality
   - no API / no automation path

## Current bias
Prefer tools that add one of these missing capabilities:
- clean economic calendar
- breadth / internals
- options / OI / unusual flow
- journaling / execution review
- cross-asset regime context

Avoid tools that are only another charting surface unless they add unique data.
